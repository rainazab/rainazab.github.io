the problem

the system is broken

California's bottle redemption industry exploits the workers who make it run - while hundreds of millions in value gets thrown away every year.

$15

poverty wages

Collectors walk 5+ miles daily through trash for $15 in 6 hours. No benefits, no stability, no way forward.

$600M

wasted value

40% of bottles are never redeemed. Restaurants dump hundreds weekly while collectors search empty-handed.

zero

worker power

No voice, no organization, no bargaining power. An invisible workforce left behind by a system not built for them.

the app

collector view host view live map onboarding

instant job claiming

Collectors see and claim available jobs in real time from the live map - no refresh, no delay.

live earnings dashboard

Completed jobs, bottle totals, and earnings all visible in one transparent view.

route optimization

Nearby jobs surface first so collectors maximize value per trip.

photo verification

Photo-backed completion builds trust between hosts and collectors and keeps records clean.

role-based onboarding

Tailored sign-in for collectors and hosts - so each user sees only what they need.

impact tracking

Bottles diverted, CO2 saved, and community activity tracked and shown in-app.
